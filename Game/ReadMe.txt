HOW TO USE
- Keyboard:
	- T : Slows down the game to 1/10 of game time while holing 'T'.
	- P : Pause the game.
	- W or Up Arrow : Accelerate in the direction the ship is pointing.
	- A or Left Arrow : Rotate left.
	- D or Right Arrow : Rotate Right.
	- I : Adds more asteroids to the screen.
	- O : Removes random asteroids.
	- N : Spawn a new ship after dying due to asteroid collision.
	- Spacebar : Fire a bullet.
	- S : If you grabbed a starburst, you can activate it with 'B'.
